## Setup
Install all the node modules
```
~> yarn
~> cd test/e2e/test && yarn
```

## Build
Build this thing (driven by rollup)
```
~> yarn build
```

### Test
**Make sure you build first**

Unit tests (Karma & Jasmine)
```
~> yarn test
```

e2e tests (nightwatch)
```
~> yarn test_e2e
```

How to set up this bad boy

Typester can be installed via NPM or Yarn…

{% highlight shell %} $ npm install typester-editor --save # or $ yarn add typester-editor --save {% endhighlight %}

… and then instantiated in your code:

{% highlight javascript %} import Typester from 'typester-editor'; // or var Typester = require('typester-editor'); new Typester({ el: document.querySelector('[contenteditable]') }); {% endhighlight %}
For more info check out our docs

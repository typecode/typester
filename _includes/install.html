How to setup this bad boy

Via NPM/Yarn

{% highlight shell %} $ npm install typester --save # or $ yarn add typester --save {% endhighlight %}

And then in your code

{% highlight javascript %} import Typester from 'typester'; // or var Typester = require('typester'); new Typester({ el: document.querySelector('[contenteditable]') }); {% endhighlight %}

And that's that! (for now)

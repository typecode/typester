    AppContainer

    The top most container for the Typester app stack. This container sets up the FormatterContainer, UIContainer, and CanvasContainer containers which in return setup various modules that make up the app stack.

    Return:

    appContainer

    AppContainer instance

    Example:

    new AppContainer({
  dom: {
    el: domElement
  }
});
